Scanner Example

Scanning objects directly creates interesting textures and patterns.

Try materials like fabric, plastic wrap, or translucent papers.

Scanning is a good option if you can't access a photocopier.

Photocopier Example

Moving images during copying creates dynamic distortion effects.

Try the techniques suggested above to achieve different results.

Great for working with your existing project images.

Five Easy Methods

You can use photocopier OR scanner:

1. Movement Blur

  1. Put image on photocopier
  2. Start copying
  3. Slowly move image

2. Open Lid Effects

  1. Put image on photocopier
  2. Keep lid partly open
  3. Make copy

3. Material Overlay

  1. Put image on photocopier
  2. Add fabric or plastic on top
  3. Make copy

4. Copy of Copies

  1. Copy your image once
  2. Copy that copy again
  3. Repeat 2-3 times

5. Scan Objects

  1. Put objects on scanner
  2. Arrange for good composition
  3. Try different textures
  4. Layer materials
