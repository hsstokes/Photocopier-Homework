Five Easy Methods to Try

You can use both photocopier techniques with printed images and try scanning objects directly:

1. Movement Blur

  1. Put image on photocopier
  2. Start the copy
  3. Slowly move the image

2. Open Lid Effects

  1. Put image on photocopier
  2. Keep lid partly open
  3. Copy with light coming in

3. Material Overlay

  1. Put image on photocopier
  2. Put material on top (fabric, plastic)
  3. Make the copy

4. Copy of Copies

  1. Copy your image once
  2. Copy that copy again
  3. Repeat 2-3 times

5. Scan 3D Objects

  1. Place interesting objects directly on scanner
  2. Arrange them for interesting compositions
  3. Try objects with varied textures (fabric, mesh, paper)
  4. Experiment with layering transparent materials
